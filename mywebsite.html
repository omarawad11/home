MarvelBot - AI-Powered University Class Registration Assistant ChatBot

Designed and deployed an intelligent chatbot solution to streamline the class registration process at the University of Jordan. The project addressed challenges such as limited seat availability and conflicting schedules for students.

The chatbot, built using Python, TensorFlow, and a large language model, provides real-time class seat availability, automated email notifications, and a proprietary seat trading platform via a Telegram bot interface. I managed the entire project lifecycle from conceptualization and system design to robust deployment and user acceptance testing.

The pilot phase involved 200 participants and processed 4,378 API requests, significantly enhancing registration efficiency and student satisfaction. Key technologies utilized include SQL databases, Selenium for testing, and virtual machines for scalable software deployment.
